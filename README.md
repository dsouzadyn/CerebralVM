# Cerebral VM

Cerebral is a brainfuck VM written in C++. It can run any possible brainfuck program.

## Usage

Tested on 64-bit Arch Linux running kernel 5.0.10 and g++ version 8.3.0.

### Compiling

```sh
$ make
```

### Using

```sh
$ ./cerebral demos/hello.bfk
```

### Cleaning up

```sh
$ make clean
```
